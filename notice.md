# MakeLens 공지사항

게시일: 2026-03-19
## 1. 429에러 안내
분당 할당량 또는 일일 할당량 만료로 뜨는 429에러.
'어? 저는 이제 막 쓰기 시작했는데 떠요. 일일 할당량이 다되었다뇨?'
아뇨!!! '분당 할당량' 문제가 제일 큽니다.
분당은, 1분당이 아닌 N분당 입니다.
그래서 1분/5분/15분/40분 알 수 없기에, 
과부하 에러인 503/타임아웃과 같다고 보시면 됩니다.
해결책 : 다른 모델 사용 또는 재시도 (flow 또는 다른모델을 이용권장)

## 2. 메이크렌즈 v1.6.1 업데이트!
1) 썸네일 제작 Flow 추가
: 썸네일은 기존 나노2로만 제작되었습니다. 하지만 과부하가 생기면 ㅠㅠ
그래서 Flow를 추가! 
다만, Flow방식은 API방식과 달라서 스타일/퀄리티가 낮을 수 있습니다.
(반드시 MakelensAuto 확장앱 재설치 | 크롬창 1개만 띄우고, 확장앱 꼭 켜주세요)

2) GEMINI TTS 사용시 AI 오타 자동수정
: 위스퍼 ai의 잘못된 오타들을 중간에 한번 gemini가 오타수정을 하여,
자막수정 소모시간을 더욱 줄여줍니다.

3) 부메랑 모드 선택
: 이제 비디오재생 기본은 '반복재생' 입니다. 원하실때 부메랑 토글을 켜주세요.

4) 스타일 프롬프트
: 드래그앤드롭 가능 및 이미지가 아닌 텍스트입력만으로도 가능합니다.

5) 무료 GEMINI API 사용가능
: 이제 FLOW 덕분에, 무료 GEMINI API를 사용하는것도 가능합니다.
무료 API는 오직 분석/대본생성/분할 등 텍스트 관련 된 항목에 사용되며,
총 요청이 하루 10번이기때문에, 버텍스를 우선 권장합니다 :)
(즉, 무료gemini api + flow + grok으로 적지만 완전 무료로도 영상생성 가능 / 썸네일 flow)

6) 그록오토->메이크렌즈 오토 변경
: 이제 확장앱 이름이 'GrokAuto'가 아닌 'MakelensAuto'로 변경되었습니다.
시작은 그록연동이었으나, 현재 위스크, 플로우 연동까지 담당하여 유저분들의 혼동을 없애기위해
확장앱 이름을 변경하였습니다.

7) 기타 다양한 버그 수정

기타 사용법 및 자세한 내용은
프로그램 좌측 '물음표'를 눌러
챗봇에게 물어보세요 :)

## 롱폼관련 추가 안내
- 메이크렌즈는 Gemini api 기반이며, 로컬 프로그램입니다. 
롱폼은 쇼츠와 달리 대본이 긴 만큼, 많은 api 요청이 따르고, 인터넷 환경과 컴퓨터 사양이 너무 중요합니다.
대본이 길어 여러차례 gemini 요청중 1번이 끊기면 실패로 돌아가기도하고, 와이파이로 인해 인터넷환경이 좋지 않을시, 에러가 생길수도 있으며,
컴퓨터 사양이 낮은 CPU와 RAM 사용시, 시간이 엄청 오래걸리거나, 부하가 올 수 있습니다.
최대한 개선중이며, 위 내용을 꼭 인지하시고 스트레스가 덜 하시길 바라봅니다.
업그레이드는 계속 됩니드아~!!!
